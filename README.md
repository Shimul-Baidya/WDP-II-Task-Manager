# Task Management API

A RESTful API for managing tasks built with Node.js and Express.js. This project provides endpoints to create, read, update, and manage tasks with priorities and completion status.

## Features

- ✅ Get all tasks with details (id, title, completed, priority, createdAt)
- ✅ Get individual task by ID
- ✅ Health check endpoint
- ✅ Error handling for invalid requests
- ✅ Organized route structure using Express Router

## Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)
- Postman (for API testing)
